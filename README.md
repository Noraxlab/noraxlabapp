# NORAXLAB - Cyber Elite Content Platform

A modern Flutter application with Firebase backend.

## Features
- Modern Cyber Theme with animations
- Firebase Realtime Database integration
- User & Admin separate apps
- Content browsing and downloading
- Admin content management

## Setup
1. Flutter install করুন
2. Firebase project তৈরি করুন
3. `flutter pub get` রান করুন
4. `flutter run` রান করুন

## Admin Access
Default: admin@noraxlab.com / Admin@123
